# Roberto San Miguel Portfolio

Dark modern portfolio website built with React and Vite.

## Local setup

```bash
npm install
npm run dev
```

## Build test

```bash
npm run build
```

## Deploy

This repository includes a GitHub Pages workflow at:

```text
.github/workflows/deploy.yml
```

The repository name should be:

```text
rsanmiguel2023.github.io
```

The website will publish to:

```text
https://rsanmiguel2023.github.io/
```
